API documentation


Parameter Explanation
users_list Twitter user names separated by ';'. No need to add twitter's '@'. There is no cap to user count.
TS TS stands for time-series. Parameter can take values 0 or 1. If 0 returns not a time-series object. If 1 returns time-series object. For now, it automatically returns hourly data. No other functionalities are present.
Count Number of tweets to retrieve from each user. Maximum value of parameter is 200.
An example http://.../api?user_list=barackobama;RealDonaldTrump&count=50&TS=1
